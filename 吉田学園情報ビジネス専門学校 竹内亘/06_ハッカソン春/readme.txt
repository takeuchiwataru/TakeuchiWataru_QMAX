	タイトル : 写真泥棒

	使用言語 : C言語
	ジャンル : アクションゲーム
	動作環境 : Windows10
	製作人数 : 4人
	テーマ   : 映す

	■操作方法
	[キーボード]
	W,A,S,D : 移動
	SPEACE  : ジャンプ
	ENTER   : 映す
	↑←↓→: カメラ
	
	ENTER   : 決定
	P       : ポーズ
	
	[ゲームパッド(Logicool)]
	左スティック,十字キー : 移動
	Aボタン               : ジャンプ
	Bボタン               : 映す
	STARTボタン           : ポーズ
	
	* DirectInputで動きます
