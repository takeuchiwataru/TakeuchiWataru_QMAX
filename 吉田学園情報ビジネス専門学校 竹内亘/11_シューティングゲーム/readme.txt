	タイトル : カメの大逆襲
	制作期間 : 2018年6月〜2018年9月
	使用言語 : C++
	ジャンル : シューティングゲーム
	動作環境 : Windows10
	
	■操作方法
	[キーボード]
	1P
	W,A,S,D : 移動
	SPEACE  : 弾の発射
	F       : 必殺技
	
	2P
	↑←↓→: 移動
	ENTER   : 弾の発射
	M       : 必殺技

	共通
	ENTER   : 決定
	P       : ポーズ
	
	[ゲームパッド(ELECOM)]
	左スティック,十字キー : 移動
	Aボタン               : 弾の発射
	Bボタン               : 必殺技
	STARTボタン           : ポーズ
	
	* XInputで動きます。
