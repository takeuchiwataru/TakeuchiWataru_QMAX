	タイトル : MonkeyCatch
	制作期間 : 2018年11月〜2019年2月
	使用言語 : C++
	ジャンル : 3Dアクションゲーム
	動作環境 : Windows10
	
	■操作方法
	[キーボード]
	W,A,S,D : 移動
	Q,E     : カメラ
	J       : ジャンプ
	ENTER   : 虫網
	P       : ポーズ
	
	[TUTORIAL]
	1でSTART
	
	[ゲームパッド(ELECOM)]
	左スティック,十字キー : 移動
	右スティック          : カメラ
	Aボタン               : ジャンプ
	Bボタン               : 虫網
	STARTボタン           : ポーズ
	
	* DirectInputで動きます。
