	タイトル : 接待ゴルフ

	動作環境 : Windows10
	製作人数 : 4人
	テーマ   : 穴
	
	■操作方法
	[キーボード]
	SPEACE  : ゴルフボールを打つ
	W,A,S,D : 穴の移動
	ENTER   : 決定
	P       : ポーズ
	
	[ゲームパッド(ELECOM)]
	左スティック          : 穴の移動
	Aボタン               : ゴルフボールを打つ
	
	* DirectInputで動きます。
