	タイトル : リペア・メモリー
	制作期間 : 2017年11月〜2018年3月
	使用言語 : C言語
	ジャンル : 3Dパズルアクションゲーム
	動作環境 : Windows10
	製作人数 : 5人
	
	■操作方法
	[キーボード]
	● 探索モード
	↑←↓→        : 移動
	SHIFT + ↑←↓→: ダッシュ
	Q,E             : カメラ回転
	ENTER           : 決定
	P               : ポーズ
	
	● パズルモード
	↑←↓→        : パズル移動
	SHIFT           : パズル変更
	ENTER           : パズルはめる
	P               : ポーズ
	
	[ゲームパッド(PS4)]
	● 探索モード
	左スティック,十字キー : 移動
	R2長押し              : ダッシュ
	〇ボタン              : 決定
	×ボタン              : キャンセル
	△ボタン              : ポーズ
	右スティック          : カメラ移動
	R1,L1                 : カメラ移動
	L2                    : カメラリセット
	
	● パズルモード
	左スティック,十字キー : パズル移動
	〇ボタン              : パズルはめる
	×ボタン              : キャンセル
	△ボタン              : ポーズ
	□ボタン              : パズル変更
